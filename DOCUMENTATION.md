## Layout

- **`docs/README.md`** — Main documentation index (setup, training, deployment, evaluation, reference).
- **`docs/*.md`** — Topic docs (hardware, dataset, env vars, model naming, optimization, export, evaluation, testing).
- **`wiki/*.md`** — Workflow and reference (Getting Started, Configuration, Training, Evaluation, Deployment, Makefile, Scripts, Troubleshooting).
- **Root `*.md`** — Checklists and short guides (README, QUICK_GUIDE, PIPELINE, POST_TRAINING, DEPLOYMENT, OPTIMIZATION_QUICKSTART, etc.).
